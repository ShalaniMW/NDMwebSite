Question Forum

Understanding the design and management issues associated with the operation and control of TCP/IP networks. The aim of this unit is to provide in-depth treatment on effective usage of active and passive monitoring/management tools and techniques, such as Network Mapping, network monitoring, Server Monitoring, the Simple Network Management Protocol (SNMP), application-layer logging and the Lightweight Directory Access Protocol (LDAP). The Unit will also cover such issues as fault tolerance, disaster recovery, network security, and HTTP issues.
